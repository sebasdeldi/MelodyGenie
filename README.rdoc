== README

* Application, developed using Ruby on Rils and the Spotify API
* The app lets you discover new artists, based on an artist that you alredy like, lesten to a preview of their top tracks, and then allows you to search for a free download for the song or listen the whole album freely on spotify.

About the app:

* Ruby version 2.2.1
* Rails version 4.2.5
* FrontEnt framework Semantic-ui
